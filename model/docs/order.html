A standard package can be used to group any placements in a cost strategy together. Cost will be attributed identically across all impressions within the Package, regardless of which placement received the impression.
enumeration roadblock
documentation
A roadblock placement is typically a group of placements that are all served on a webpage together. Cost information will be associated whenever any or all placements in the Roadblock receive an impression.
